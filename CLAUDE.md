# AdiumY — Adium fork

This is a fork of the upstream [Adium project](https://github.com/adium/adium).

## Upstream comparison

The tag `mark-final-upstream` points to the last commit from upstream before this fork diverged. Use it as a shortcut for comparing against the original codebase:

- `git diff mark-final-upstream -- <path>` — see what changed since the fork
- `git log mark-final-upstream..HEAD` — see all fork commits
- `git merge-base mark-final-upstream HEAD` — find the fork point
